Welcome to LTT Help Centre!


- To search for an admin, enter your search query in the search bar then proceed to click on the “Search” button.
- To add an admin, click on the “Add Admin” button that will navigate you to the add admin screen.
- To edit an admin, click on the “Edit” button beside the admin you want to edit. The system will navigate you to the edit admin screen.
- To remove an admin, click on the remove button then confirm the removal.


If you can not add, edit or remove an admin, please follow these steps:
  1. Refresh the webpage.
  2. Log out of the system and then log back into the system.
If the admin can’t be removed, please do the following:
  1. Ensure that you clicked the “Yes” button on the confirmation modal.
  2. Make sure that you are connected to the internet.
  3. Log out of your account and then proceed to log back in afterwards.

If the issue still persists, please contact our support team at hanru@limtt.co.za.
